Documentación de la API

Base URL: http://localhost:8000/api/v1
Endpoints de Búsqueda
GET /buscar/{ruc} Buscar por RUC
GET /buscar/nombre/{nombre} Buscar por nombre
GET /buscar/departamento/{departamento} Buscar por departamento
GET /buscar/estado/{estado} Buscar por estado
Endpoints de Sistema
GET /estadisticas Obtener estadísticas
GET /departamentos Listar departamentos
GET /estados Listar estados
Ejemplos de Uso
Ejemplo 1: Buscar RUC
GET /api/v1/buscar/10452159428

Respuesta:
{
    "success": true,
    "data": {
        "ruc": "10452159428",
        "nombre_razon_social": "GARCIA CHANCO CARLOS AUGUSTO",
        "estado_contribuyente": "ACTIVO",
        ...
    }
}
Ejemplo 2: Buscar por nombre
GET /api/v1/buscar/nombre/EMPRESA?limit=10

Respuesta:
{
    "success": true,
    "count": 10,
    "data": [...]
}
